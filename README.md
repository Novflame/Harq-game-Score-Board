# Harq-game-Score-Board
جدول تسجيل النقاط لعبة حريق
🔥 Harq Score Tracker

A simple web app to track scores for the Harq card game.

📖 Description

This project helps players track their points during Harq matches in an easy and clean way.

🚀 Features

- Add multiple players
- Track scores
- Mobile-friendly design

🛠️ Technologies

- HTML
- CSS
- JavaScript

▶️ Usage

Open the app, add players, and start tracking scores.
